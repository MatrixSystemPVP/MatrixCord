# Documentation
Welcome to the documentation of MatrixCord!

## Contributing
You are welcome to contribute to the package at any time. Got some ideas or just want to edit a misspelling in the documentation? Head over to the GitHub repository via [this](https://github.com/MatrixSystemPVP/MatrixCord/tree/dev) link or via the page in the documentation you want to edit.

## Help
If you don't understand something in the documentation, you are experiencing problems, or need some other help for the package, please don't hesitate to join our [discord Server](https://discord.gg/kEbQyy6).
